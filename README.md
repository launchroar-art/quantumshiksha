# Quantum Shiksha

Static website for Quantum Shiksha Pragati Foundation, designed for GitHub Pages.

## Files

- `index.html` contains the full website structure.
- `style.css` contains the responsive design system and layout.
- `script.js` handles the mobile menu and scroll reveal behavior.
- `assets/images/products` contains curated product photos.
- `assets/images/community` contains education and outreach photos.
- `assets/images/showcase` contains featured homepage visuals.
- `CNAME` keeps the custom domain configuration for `quantumshiksha.org`.

## Deploy

This repository is ready to publish through GitHub Pages from the root of the
default branch.
